# CodeDNA – Biomorphic Code Visualization Engine

CodeDNA is a conceptual software system that transforms Java source code into a biological-style visual representation. Instead of viewing code as plain text, this project interprets program structure as a living organism, where each part of the code contributes to the shape, behavior, and complexity of a generated creature.

The system analyzes a given Java file, extracts its structural elements, and converts them into a dynamic visual form using SVG-based rendering. This approach provides a more intuitive and engaging way to understand code complexity and organization.

The program is designed to run simply: the user places a Java source file in the sample input, executes the system, and a visual organism is automatically generated as output.

---

## Features

* Converts Java source code into a visual biological organism
* Represents program structure in an intuitive graphical form
* Maps code elements directly to creature features
* Generates unique output for every different input file
* Uses lightweight technologies (Java + SVG + HTML)
* Requires minimal setup and runs automatically with input file
* Helps visualize complexity, structure, and code quality

### Creature Behavior Mapping

* More methods → More tentacles
* Larger code size → Bigger organism body
* More loops → Repetitive or dynamic motion elements
* More conditions → Branching structures
* Deep nesting → Mutated or complex internal patterns
* More comments → Balanced and stable appearance
